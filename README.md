## 🎯 Objectif du projet

Dans le cadre du module de Génie Logiciel (Bloc 2), ce projet a pour objectif d’analyser un code source fourni sans documentation.

Les objectifs sont de :

* Comprendre un code existant
* Identifier ses forces et ses faiblesses
* Documenter son fonctionnement
* Proposer des améliorations
* Développer le travail en équipe

---

## 🧾 Description du projet

Le projet étudié est un site web vitrine représentant une agence fictive appelée **DevStart Agency**.

Ce site permet de :

* présenter les services d’une agence web
* valoriser son expertise
* rassurer les clients
* proposer un moyen de contact

### ⚙️ Technologies utilisées

* HTML
* CSS

⚠️ Le site est **statique** :

* pas de JavaScript
* pas de backend
* aucune interaction dynamique

---

## 🧱 Structure du site

Le site est organisé en plusieurs sections :

* Navbar (menu de navigation)
* Hero (présentation principale)
* Services
* À propos
* Témoignages
* Contact
* Footer

Chaque section est construite principalement avec des `<div>` et stylisée via CSS.

---

## 📂 Structure du dépôt

```
/original
    → Code source initial (non modifié)

/annotated
    → Code commenté (HTML + CSS)

/README.md
    → Documentation du projet
```

---

## ⚠️ Problèmes identifiés

* ❌ Noms de classes non explicites (ex : `.b1`, `.c`, `.d`)
* ❌ Absence de commentaires dans le code original
* ❌ HTML non sémantique (usage excessif de `<div>`)
* ❌ Formulaire de contact non fonctionnel
* ❌ Site non responsive
* ❌ Absence de JavaScript
* ❌ Répétition du code CSS
* ❌ Mauvaise organisation globale
* ❌ Accessibilité ignorée
* ❌ Liens non fonctionnels
* ❌ Absence de backend

---

## 💡 Propositions d’amélioration

* ✔️ Renommer les classes avec des noms explicites
* ✔️ Ajouter des commentaires dans le code
* ✔️ Utiliser des balises sémantiques (`<header>`, `<section>`, etc.)
* ✔️ Rendre le site responsive (media queries)
* ✔️ Ajouter du JavaScript pour les interactions
* ✔️ Implémenter un backend pour le formulaire
* ✔️ Optimiser le CSS
* ✔️ Améliorer l’accessibilité
